Data Handlers
===============

The Data Handlers reference contains detailed descriptions of the different classesused to pre-process data, including input data (LBA), Ontology Database (LION) and Lipid Reaction Networks explorer (Linex).


.. note::

    This API reference only contains end-user documentation.
    If you are looking to hack away at data handlers' internals, you will find more detailed comments in the source code.
